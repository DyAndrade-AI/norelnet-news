# Norelnet News

Plataforma web de noticias desarrollada con **MongoDB**, **Redis** y **Cassandra**.
Proyecto colaborativo de la Universidad de Guanajuato.

## Descripción
El sistema mostrará noticias recientes con almacenamiento distribuido:
- **MongoDB**: base de datos principal de noticias y usuarios.
- **Redis**: caché para optimizar consultas y manejos de sesión.
- **Cassandra**: registro histórico y analítica de noticias más vistas.
